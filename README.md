# 🍔 Eatly

Eatly is a mood-based food recommendation web app that helps users discover restaurants based on how they’re feeling. Instead of endlessly scrolling, Eatly matches your mood to the perfect place to eat.

## ✨ Features
- Mood-based restaurant recommendations
- Clean and responsive user interface
- Modern icons and styling
- Cloud-powered backend services

## 🧰 Technologies Used
- Vite
- JavaScript
- Foursquare API
- React
- React Icons
- React Bootstrap
- Firebase
- HTML
- CSS
